# Handling conditions

[if](https://docs.python.org/3/reference/compound_stmts.html#the-if-statement) 문을 사용하여 분기(Condition) 명령을 처리합니다.

`if` 구문

```python
if expression:
    # 실행할 코드
else:
    # 실행할 코드
```

[비교(comparison) 연산자](https://docs.python.org/3/library/stdtypes.html#comparisons)

- < 미만
- < 초과
- == 같음
- \>= 이상(같거나 큼)
- <= 이하(같거나 작음)
- != 같지 않음
