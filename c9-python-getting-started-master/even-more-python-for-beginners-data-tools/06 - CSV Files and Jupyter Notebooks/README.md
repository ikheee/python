# CSV 파일과 Jupyter Notebooks

CSV 파일은 쉼표로 구분 된 변수 파일입니다. CSV 파일은 데이터를 저장하는 데 자주 사용되며, Jupyter Notebook에서 CSV 파일의 데이터에 액세스하려면, 파일을 업로드 해야합니다.
