# 상속(Inheritance)

[상속](https://docs.python.org/3/tutorial/classes.html#inheritance)은 다른 클래스의 모든 메서드와 속성을 상속하는 클래스에서 정의해 사용할 수 있습니다. 부모(parent) 또는 기본(base) 클래스는 상속되는 클래스입니다. 자식(child) 또는 파생(derived) 클래스는 다른 클래스에서 상속하는 클래스입니다.
