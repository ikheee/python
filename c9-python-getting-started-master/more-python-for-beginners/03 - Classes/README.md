# 클래스(Classes)

[클래스](https://docs.python.org/3/tutorial/classes.html)는 데이터 구조와 동작을 정의합니다. 클래스를 사용하면 데이터와 기능(functionality)를 함께 그룹화 할 수 있습니다.
